## 📖 About

Notes App is a Single Page Application (SPA) developed with **Angular 19** and **Angular Signals** that allows users to create, edit, organize and manage personal notes through a clean and intuitive interface.

This project was created to strengthen my understanding of Angular's modern reactive programming model using **Signals**, while applying Angular best practices such as component-based architecture, standalone components and service-based state management.

All notes are stored locally using the browser's **Local Storage**, allowing data persistence without requiring a backend.

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate to the project folder

```bash
cd NotesAppSignals
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
ng serve
```

Open your browser at:

```
http://localhost:4200
```
