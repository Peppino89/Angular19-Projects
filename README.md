# Angular Projects

## About

This repository contains a collection of Angular 19 projects developed during my learning journey.

The main goal of these projects is to improve my knowledge of Angular by building practical applications and exploring both the traditional Angular approach based on **RxJS and Observables** and the new **Signals API** introduced in the latest versions of the framework.

Some projects are available in two versions:

- **Classic Version** – built using the traditional Angular architecture.
- **Signals Version** – refactored using Angular Signals to explore the new reactive programming model.

---

## Technologies

- Angular 19
- TypeScript
- HTML5
- CSS3
- Angular Signals
- RxJS
- Standalone Components
- Reactive Forms
- Angular Router
