## 📖 About

Notes App is a Single Page Application (SPA) developed with **Angular 19** that allows users to create, edit, organize and manage personal notes through a clean and intuitive interface.

This project was built to strengthen my understanding of Angular fundamentals using the traditional service-based architecture, reinforcing concepts such as component communication, application structure and local data persistence.

All notes are stored locally using the browser's **Local Storage**, allowing data to persist between sessions without requiring a backend.

- ✅ Create new notes

## 🚀 Getting Started

### Clone the repository

```bash
git clone <repository-url>
```

### Navigate to the project folder

```bash
cd NotesApp
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
ng serve
```

Open your browser at:

```text
http://localhost:4200
```
